## 🚀 Usage Instructions

### For Finding Documentation
1. Start at `docs/README.md` for overview
2. Navigate to relevant theme (auth/, filters/, etc.)
3. Read theme `README.md` for quick overview
4. Follow links to specific documents

### For Adding New Documentation
1. Choose appropriate `docs/<theme>/` folder
2. Create file with pattern: `<topic>-<context>.md`
3. Write in English only
4. Use lowercase filenames with hyphens
5. Update relevant theme `README.md`
6. Use relative paths for internal links
7. Update `docs/README.md` if needed

### For Following Links
- All internal links use relative paths
- Format: `./file.md` or `../auth/setup.md`
- No absolute paths
- No root-level references
